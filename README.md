# Usage

```txt
$ node index.js exec --help
Usage: ovmwd exec [options]

execute OVM1 withdrawals

Options:
  --hash <hash>       withdrawal transaction hash
  --pk <key>          private key to prove or relay with
  --l1-rpc <url>      l1 rpc url
  --l2-rpc <url>      l2 rpc url
  -h, --help          display help for command
```
